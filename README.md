# Sales Operations & KPI Performance Dashboard (Excel)

## Project Overview
This project simulates an operational reporting environment where transactional sales data is validated, structured, and transformed into KPI-driven insights to support business operations. It focuses on ensuring data reliability, enabling performance visibility, and supporting decision-making through structured reporting.

- **Domain:** Sales operations  
- **Tool:** Microsoft Excel  
- **Focus:** Data quality, KPI reporting, operational performance tracking  

The dataset represents multi-channel sales transactions across categories, customer segments, and regions. The project involves preparing data for reporting, ensuring consistency, and building a dashboard to monitor key operational metrics.
